# Machine Learning Project

This project implements multiple machine learning approaches for image classification.

## Project Overview

The objective of this project is to classify images using both traditional machine learning and deep learning techniques.

Implemented models include:

- Perceptron
- Linear SVM
- Kernel SVM
- Convolutional Neural Network (CNN)

## Dataset

- MNIST Handwritten Digits Dataset
- Binary Classification
- Multi-Class Classification

## Technologies

- Python
- NumPy
- Scikit-Learn
- TensorFlow / Keras
- Matplotlib
